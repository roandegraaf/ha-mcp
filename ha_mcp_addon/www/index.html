MCP Server — Connection Guide

Copy a config snippet below to connect your MCP client to this Home Assistant instance.

Server URL: http://homeassistant.local:8099/mcp/
Network access: The MCP server listens on port 8099. Your MCP client must be able to reach this address. If connecting from outside your local network, configure port forwarding or a reverse proxy with HTTPS. Some clients (like web-based ones) may block mixed HTTP/HTTPS content — use HTTPS on both ends in that case.
